# Detour Third-Party Components

`locus_unity/Editor/Detour` contains `Locus.Detour.dll`, a merged .NET assembly used only inside the Unity Editor for Locus hot reload (method redirection). It is generated by `bun run unity:bundle-detour` (`scripts/build-locus-detour-bundle.mjs`) from the official NuGet packages below; the original DLLs are not kept in this repository.
